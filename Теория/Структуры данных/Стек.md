***
***Аналогия.***
Гора посуды. Можно положить тарелку наверх и убрать ее => читается только верхний элемент и он же удаляется из стека.
- Поддерживается 2 действия: ***вставка*** и ***удаление***.
***
***Пример.***
При вызове функций компьютер сохраняет в памяти значения всех переменных для этого вызова.
```python
def greet(name):
	print("Hello, " + name + "!")
	greet2(name)
	print("Greeting ready to say bye...")
	bye()
	
def greet2(name):
	print("How are you, " + name + "?")
	
def bye():
	print("Ok bye!")
	
greet("Dora")
```
1. Компьютер выделяет блок памяти для вызова функции `greet("Dora")`. Переменной `name` присваивается значение `"Dora"`.
2. Вывод `"Hello, Dora!"`, после чего компьютер вновь выделяет блок памяти для вызова функции `greet2("Dora")`. ***ВАЖНО: `greet` не завершается, она находится в "подвешенном" состоянии.***
3. Вывод `"How are you, Dora?"`, завершение `greet2`, возврат к `greet`.
4. После вывода строки вызывается `bye` => блок для этой функции добавляется на вершину стека.
5. Выполнение `bye`, удаление ее с вершины стека, возврат к `greet`.
***
***Вывод.***
Когда мы вызываем функцию из другой функции, вызывающая функция приостанавливается в частично завершенном состоянии. При этом вызываемая функция создает блоки данных, которые помещаются на верхушку стека. При ее выполнении, данные удаляются с вершины и "штурвал" переходит к исходном вызывающей функции.
