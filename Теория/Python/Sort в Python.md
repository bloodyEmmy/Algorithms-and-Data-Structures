***
Встроенный метод ***только*** для [[Массив|массивов]]. Изменяет сам искомый массив и возвращает `None`.
- Не создает копию, в отличие от [[Sorted в Python|sorted()]], старый порядок теряется навсегда.
- Экономит память по сравнению с `sorted()`, так как не дублирует данные.
***
***Синтаксис.***
```python
list.sort(key=None, reverse=False)
```
1. `key` - функция, которая применяется к каждому элементу перед сравнением.
2. `reverse` - флаг направления сортировки.
