#flashcards 
***
***Неупорядоченная*** коллекция уникальных ***элементов***, которую часто используют для проверки принадлежности и удаления дубликатов.
Поддерживает объединение, пересечение, разность и симметрическую разность.
***
***Свойства.***
1. Элементы не повторяются.
2. Порядок не гарантируется (нет индексов, как у [[Массив|массива]]).
***
***Код.***
```python
# создание
s = {1, 2, 3}
s = set([1, 2, 3, 4]) # из массива
s = set() # пустой

# работа с элементами
s.add(4) # добавить
s.remove(2) # удалить с ошибкой, если элемента нет
s.discard(2) # удалить без ошибки, если нет
s.pop() # удалить и вернуть произвольный
s.clear()

# операции с множествами
a = {1, 2, 3}
b = {2, 3, 4}
a | b # объединение {1, 2, 3, 4}
a & b # пересечение {2, 3}
a - b # разность {1}
a ^ b # симметрическая разность {1, 4}
```
