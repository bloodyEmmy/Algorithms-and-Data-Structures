***
Встроенная функция, которая возвращает ***новый*** отсортированный список из любого итерируемого объекта.
- Исходный объект не изменяется.
- В среднем работает за время [[O-большое|O(n * log2 n)]].
***
***Синтаксис.***
```python
sorted(iterable, key=None, reverse=False)
```
1. `iterable` - что сортируем ([[Массив|массив]], строка, кортеж, [[Словарь в Python|словарь]] и др.)
2. `key` - функция, которая применяется к каждому элементу до сравнения (для кастомной сортировки).
3. `reverse` - `True` для сортировки по убыванию.
***
***Пример.***
```python
sorted("cba") # ['a', 'b', 'c']
sorted((3, 1, 2)) # [1, 2, 3]
d = {'b': 2, 'a': 1}
sorted(d) # ['a', 'b']

# по длине
words = ["apple", "banana", "kiwi"]
sorted(words, key=len) # ['kiwi', 'apple', 'banana']
```
