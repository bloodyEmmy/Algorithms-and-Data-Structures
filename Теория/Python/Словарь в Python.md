#flashcards 
***
Неупорядоченная структура данных, которая хранит пары `"ключ:значение"` и позволяет быстро находить нужную информацию по ***ключу***.
Создается с помощью фигурных скобок `{}`, элементы разделяются запятыми.
```python
empty_dict = {}
student = {'name': 'Bob', 'age': 21, 'course': 3}
mix = {'fruit': 'apple', 1: [4, 6, 8]}
```
- ***Ключами*** могут быть строки, числа и кортежи. Списки и другие словари не могут быть использованы как ключи.
- ***Значения*** могут быть любого типа, включая другие словари и списки.
***
***Основные операции.***
1. Доступ к элементу (значению) по ключу.
```python
car = {'brand': 'Lada', 'model': 'Kalina', 'type': 'garbage'}
print(car['brand']) # Lada
```
2. Добавление и изменение элементов.
```python
car['color'] = 'black' # Добавил
car['type'] = 'shit' # Изменил
```
***
***Методы.***
1. `len(dict)` - количество элементов.
2. `dict.keys()` - получить все ключи.
3. `dict.values()` - получить все значения.
4. `dict.items()` - получить пары (ключ, значение).
5. `dict.get("value")` - проверить, есть ли ключ `"value"` в словаре (`True` / `False`).
