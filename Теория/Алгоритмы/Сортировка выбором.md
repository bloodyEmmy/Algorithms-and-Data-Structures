***Пример.***
Рассмотрим сортировку по убыванию. 
1. Проходимся по искомому набору элементов.
2. Находим среди них наибольший.
3. Кладем в новый список.
4. Повторяем для оставшегося набора до конца.
Все это требует [[O-большое|O(n * n)]] или `O(n ** 2)`.
***
***Реализация.***
Здесь наоборот сортируем по возрастанию.
```python
# поиск наименьшего элемента
def findSmallest(arr):
	smallest = arr[0]
	sallest_index = 0
	for i in range(1, len(arr)):
		if arr[i] < smallest:
			smallest = arr[i]
			smallest_index = i
	return smallest_index
	
# сортировка выбором
def selectionSort(arr):
	newArr = []
	for i in range(len(arr)):
		smallest = findSmallest(arr)
		newarr.append(arr.pop(smallest))
	return newArr
```
