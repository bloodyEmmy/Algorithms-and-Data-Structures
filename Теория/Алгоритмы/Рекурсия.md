Вызов функцией самой себя.
В любой рекурсивной функции необходимо указывать, в какой момент ей стоит прервать рекурсию => 
- Любая такая функция состоит из ***базового*** и ***рекурсивного*** случаев. Во втором она вызывает сама себя, в первом - не вызывает для предотвращения зацикливания.
- Рекурсивные функции имеют [[Стек]] вызовов.
***
***Пример.***
```python
def countdown(i):
	print(i)
	if i <= 1:
		return
	else:
		countdown(i - 1)
		
countdown(3) # 3 2 1
```
***
***Пример.***
```python
# факториал
def fact(x):
	if x == 1:
		return 1
	else:
		return x * fact(x - 1)
```
