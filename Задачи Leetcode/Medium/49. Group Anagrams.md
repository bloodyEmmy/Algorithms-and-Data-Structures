Дана последовательность строк `strs`, сгруппируйте анаграммы вместе. Вы можете вернуть ответ в **любом порядке**.
***Пример.***
Input: `strs = ["eat","tea","tan","ate","nat","bat"]`
Output: `[["bat"],["nat","tan"],["ate","eat","tea"]]`
#flashcards 
***
Решал задачу при помощи классических [[Хеш таблица|хеш-таблиц]], используя [[Словарь в Python|словарь]] в `Python`.
Если `n` - количество строк в `strs`, а `k` - их средняя длина, то [[O-большое|итоговая сложность]] составит `O(k * log2 k) * n = O(n * k log2 k)`.
***Решение.***
```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        hashf = {}
        for i in strs:
            i_sort = "".join(sorted(i))
            if i_sort not in hashf:
                hashf[i_sort] = [i]
            else:
                hashf[i_sort] += [i]
        return list(hashf.values())
```
1. Проходимся по каждой строке из `strs`.
2. Сортируем ее при помощи [[Sorted в Python|sorted()]], используем сортированное значение как ключ словаря `hashf`.
3. Если ключа нет, то добавляем его и в массив значений добавляем первый элемент. Если ключ есть, то добавляем строку `i` в него.
4. Выводим все значения в формате списка при помощи `list()`.
