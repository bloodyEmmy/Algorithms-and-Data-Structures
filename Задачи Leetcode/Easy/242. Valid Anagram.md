Даны две строки `s`и `t`, верните значение `true`, если `t`является строкой-анаграммой`s` и `false` в других случаях.
***P.S.*** Анаграмма — это слово или фраза, образованная путем перестановки букв другого слова или фразы с использованием всех исходных букв ровно один раз.
#flashcards 
***
Мое решение основано на использовании [[Хеш таблица|хеш-таблицы]], реализованной в `Python` в виде [[Словарь в Python|словаря]]. Не самое чистое, но понятное.
Работает за время [[O-большое|O(n)]] (Вообще, строго говоря, `O(3 * n)`, но константы опускаются).
```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        hashf = {}
        for i in s:
            if i in hashf:
                hashf[i] += 1
            else:
                hashf[i] = 1
        for i in t:
            if i not in hashf:
                return False
            else:
                hashf[i] -= 1
        l = list(hashf.values())
        for i in l:
            if i != 0:
                return False
        return True
```
1. Заводим словарь. Проходимся по первой строке и ведем подсчет количества каждой буквы в строке `s`. Записываем значения в словарь `hashf`, где ключ - буква, а значение - количество ее повторов в строке.
2. Проходимся по второй строке `t`, и считаем повторы каждой буквы. Если буквы в словаре `hashf` нет, то возвращаем `False`. Иначе уменьшаем количество повторов буквы на 1.
3. Наш ожидаемый результат: все значения в словаре равны 0. Если есть хотя бы один не равный 0, то возвращаем `False`.
