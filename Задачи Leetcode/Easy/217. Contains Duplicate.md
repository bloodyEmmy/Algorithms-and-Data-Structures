Дана целочисленная матрица `nums`. Верните значение, `true`если какое-либо значение встречается в матрице **хотя бы дважды**, и значение `false`, если каждый элемент уникален.
#flashcards 
***
Для решения используется [[Хешсет]] и его реализация на `Python` в виде [[Множество в Python|множества]].
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        hashset = set()
        for value in nums:
            if value in hashset:
                return True
            hashset.add(value)
        return False
```
1. Проходимся по всей матрице `nums`, если элемента в `set` нет, то добавляем его туда.
2. Если же элемент уже есть, то это повтор. Сразу возвращаем `True`.
3. Выполняется за время [[O-большое|O(n)]].
